OpenEdu-SerialConnect ist ein Projekt, um unter Java Daten per RS232 (serielle Schnittstelle)
auszulesen und diese auf der Konsole auszugeben, bzw. weiter zu verarbeiten

To report a bug or a make a suggestions, mail to:
kbay@gmx.de

INSTALLATION
Java 1.6 or higher has to be installed.
RXTX is also needed. Get it under: http://www.jcontrol.org/download/rxtx_de.html

CREDITS
OpenEdu-SerialConnect is an open source project, supported presently by Residuen.

OpenEdu-SerialConnect uses Java 1.6
